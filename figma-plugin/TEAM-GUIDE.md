# 🎨 PC↔모바일 자동화 플러그인 - 팀원용 설치 가이드

## 설치 방법 (3분 소요)

### Step 1: 폴더 저장
공유받은 `figma-plugin` 폴더를 컴퓨터 아무 곳에나 저장하세요.
(예: 바탕화면, 문서 폴더 등)

### Step 2: Figma Desktop 실행
⚠️ **반드시 데스크톱 앱**을 사용하세요! (웹 버전 안됨)

아직 없다면 → https://www.figma.com/downloads/

### Step 3: 플러그인 불러오기
1. Figma에서 아무 파일이나 열기
2. 상단 메뉴: **Plugins** → **Development** → **Import plugin from manifest...**
3. 저장한 폴더 안의 `manifest.json` 파일 선택
4. 완료!

### Step 4: 플러그인 실행
- 메뉴: **Plugins** → **Development** → **PC↔모바일 상세페이지 자동화**
- 또는: `Cmd+/` (Mac) / `Ctrl+/` (Windows) → "PC모바일" 검색

---

## 사용 방법

### 🔍 모듈 스캔
파일 내 PC/모바일 컴포넌트를 자동으로 찾습니다.

1. 플러그인 실행
2. **모듈 스캔 시작** 클릭

### 📄 페이지 생성
선택한 모듈로 PC/모바일 상세페이지를 동시에 생성합니다.

1. **페이지 생성** 탭 이동
2. 모듈 선택 (체크박스)
3. 드래그로 순서 조정
4. **페이지 생성** 클릭

### 🔄 콘텐츠 동기화
PC 내용을 모바일에 복사하거나 그 반대로 동기화합니다.

1. Figma에서 요소 선택
2. **콘텐츠 동기화** 탭 이동
3. **PC → 모바일** 또는 **모바일 → PC** 클릭

---

## 컴포넌트 이름 규칙

플러그인이 인식하려면 컴포넌트 이름에 접두사가 필요합니다:

| 버전 | 접두사 | 예시 |
|------|--------|------|
| PC | `pc_` | `/pc_usp_type1` |
| 모바일 | `mo_` | `/mo_usp_type1` |

---

## 문제 해결

**Q: 플러그인 메뉴가 안 보여요**
→ Figma **Desktop 앱**인지 확인하세요 (웹 버전 X)

**Q: 모듈이 스캔되지 않아요**
→ 컴포넌트 이름 앞에 `pc_` 또는 `mo_` 붙였는지 확인하세요

**Q: 플러그인이 작동하지 않아요**
→ 플러그인 닫고 다시 실행해보세요
