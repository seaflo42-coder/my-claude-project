# PC↔모바일 상세페이지 자동화 Figma Plugin

PC와 모바일 상세페이지 디자인 작업을 자동화하는 Figma 플러그인입니다.

## 주요 기능

### 1. 모듈 스캔
- 파일 내 PC/모바일 컴포넌트를 자동으로 감지
- 카테고리별 자동 분류 (USP, 기본정보, 네비게이션, 교육일정 등)

### 2. PC↔모바일 매핑
- PC 모듈과 모바일 모듈을 연결
- 이름 기반 자동 매핑 지원
- 수동 매핑 설정 가능

### 3. 페이지 자동 생성
- 선택한 모듈 순서대로 PC/모바일 상세페이지 동시 생성
- 드래그 앤 드롭으로 모듈 순서 변경
- Auto Layout 기반 정렬

### 4. 콘텐츠 동기화
- PC→모바일 또는 모바일→PC 콘텐츠 복사
- 텍스트, 이미지, 스타일 선택적 동기화
- 전체 페이지 일괄 동기화

## 설치 방법

### 1. 의존성 설치

```bash
cd figma-plugin
npm install
```

### 2. 빌드

```bash
npm run build
```

개발 중 자동 빌드:
```bash
npm run watch
```

### 3. Figma에서 플러그인 로드

1. **Figma Desktop** 앱 열기
2. 메뉴 → **Plugins** → **Development** → **Import plugin from manifest...**
3. `figma-plugin/manifest.json` 파일 선택
4. 플러그인이 Development 목록에 추가됨

### 4. 플러그인 실행

- 메뉴 → **Plugins** → **Development** → **PC↔모바일 상세페이지 자동화**
- 또는 `Cmd/Ctrl + /` → 플러그인 이름 검색

## 사용 방법

### 컴포넌트 네이밍 규칙

플러그인이 PC/모바일 모듈을 인식하려면 다음 네이밍 규칙을 따라주세요:

**PC 모듈:**
```
/pc_usp_type1
/PC_기본정보_type1
pc_네비게이션_type2
```

**모바일 모듈:**
```
/mobile_usp_type1
/mo_기본정보_type1
mobile_네비게이션_type2
```

### Step 1: 모듈 스캔

1. **모듈 스캔** 탭 선택
2. **모듈 스캔 시작** 버튼 클릭
3. PC/모바일 모듈이 자동으로 감지됨

### Step 2: 매핑 설정

1. PC 모듈과 대응되는 모바일 모듈을 선택
2. **매핑 저장** 클릭
3. 매핑은 Figma 파일에 저장되어 다음에도 유지됨

### Step 3: 페이지 생성

1. **페이지 생성** 탭 선택
2. 상세페이지에 포함할 모듈 체크박스 선택
3. 드래그하여 모듈 순서 조정
4. 페이지 이름 입력
5. PC/모바일 생성 옵션 선택
6. **페이지 생성** 클릭

### Step 4: 콘텐츠 동기화

1. **콘텐츠 동기화** 탭 선택
2. Figma에서 동기화할 요소 선택
3. 동기화 옵션 선택 (텍스트, 이미지, 스타일)
4. 동기화 방향 선택:
   - **PC → 모바일**: PC 콘텐츠를 모바일에 복사
   - **모바일 → PC**: 모바일 콘텐츠를 PC에 복사
   - **양방향 동기화**: 양쪽 모두 최신 상태로 동기화

## 카테고리 자동 분류

플러그인은 모듈 이름에서 다음 키워드를 인식하여 자동 분류합니다:

| 키워드 | 카테고리 |
|--------|----------|
| usp | USP |
| info, 기본정보 | 기본정보 |
| nav, 네비게이션 | 네비게이션 |
| schedule, 교육일정, 일정 | 교육일정 |
| pre, 사전설명회, 설명회 | 사전설명회 |
| header | 헤더 |
| footer | 푸터 |
| benefit, 혜택 | 혜택 |
| review, 후기 | 후기 |
| faq | FAQ |
| cta | CTA |
| price, 가격 | 가격 |
| curriculum, 커리큘럼 | 커리큘럼 |

## 문제 해결

### 모듈이 스캔되지 않아요
- 컴포넌트 이름에 `pc_`, `mobile_`, `mo_` 등의 접두사가 있는지 확인
- 컴포넌트가 Component 또는 Component Set인지 확인

### 매핑이 저장되지 않아요
- Figma 파일에 편집 권한이 있는지 확인
- 파일을 저장한 후 다시 시도

### 동기화가 안 돼요
- PC↔모바일 매핑이 올바르게 설정되었는지 확인
- 동기화할 요소가 매핑된 컴포넌트 내에 있는지 확인
