# PC↔모바일 상세페이지 자동화

Figma에서 PC와 모바일 상세페이지 디자인 작업을 자동화하는 플러그인입니다.

자세한 사용법은 [figma-plugin/README.md](./figma-plugin/README.md)를 참고하세요.

## Quick Start

```bash
cd figma-plugin
npm install
npm run build
```

Figma Desktop → Plugins → Development → Import plugin from manifest → `figma-plugin/manifest.json` 선택
